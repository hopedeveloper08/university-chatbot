# Problem Statement

Unversity students and staff frequently seek information about academic regulations, procedures and services. However, navigating lengthy documents such as university bylaws, academic calendars and administrative guidelines can be time-consuming and confusing. This project aims to develop a university chatbot system that utilizer a language model to understand student queries in persian natural language and retrieve accurate responses based solely on official university documents. The chatbot will not generate responses independently but will act as an intelligent interface between students and the structured institutional knowledge, improving accessibility and reducing administrative load.

## Project Objective

- Develop a chatbot capable of understanding persian natural language question from students.
- Implement a retrieval-based system that extracts answer strictly from official university documents (e.g., regulations, policies).
- Integrate a language model for semantic understanding and question reformulation if needed.
- Ensure the system can handle persian language.
- Deploy the chatbot as a web-based service for easy access by students and staff.
